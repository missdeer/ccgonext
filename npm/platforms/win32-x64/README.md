# @ccgonext/win32-x64

Platform-specific binary for [ccgonext](https://github.com/missdeer/ccgonext) (Windows x64).

## Overview

This package contains the pre-built binary for Windows x64. It is automatically installed as an optional dependency when you run `npm install ccgonext` on Windows x64 systems.

**Do not install this package directly.** Instead, use:

```bash
npm install -g ccgonext
```

## Supported Platforms

- Windows x64/amd64
