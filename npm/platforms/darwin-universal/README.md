# @ccgonext/darwin-universal

Platform-specific binary for [ccgonext](https://github.com/missdeer/ccgonext) (macOS Universal - x64 and arm64).

## Overview

This package contains the pre-built binary for macOS (Universal). It is automatically installed as an optional dependency when you run `npm install ccgonext` on macOS.

**Do not install this package directly.** Instead, use:

```bash
npm install -g ccgonext
```

## Supported Platforms

- macOS x64 (Intel)
- macOS arm64 (Apple Silicon)
