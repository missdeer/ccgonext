# @ccgonext/linux-x64

Platform-specific binary for [ccgonext](https://github.com/missdeer/ccgonext) (Linux x64).

## Overview

This package contains the pre-built binary for Linux x64. It is automatically installed as an optional dependency when you run `npm install ccgonext` on Linux x64 systems.

**Do not install this package directly.** Instead, use:

```bash
npm install -g ccgonext
```

## Supported Platforms

- Linux x64/amd64
