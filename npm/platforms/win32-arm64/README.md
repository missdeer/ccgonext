# @ccgonext/win32-arm64

Platform-specific binary for [ccgonext](https://github.com/missdeer/ccgonext) (Windows ARM64).

## Overview

This package contains the pre-built binary for Windows ARM64. It is automatically installed as an optional dependency when you run `npm install ccgonext` on Windows ARM64 systems.

**Do not install this package directly.** Instead, use:

```bash
npm install -g ccgonext
```

## Supported Platforms

- Windows arm64
